# Fitty Design System

Fitty uses a light, friendly, mobile-first UI. The default identity is wellness-oriented rather than clinical: warm white backgrounds, coral actions, mint health signals, lavender accents, readable dark slate text, soft surfaces, and visible focus states.

## Tokens

- Primary: pastel coral `#ff7f6e`
- Secondary: mint teal `#39c6a7`
- Accent: lavender `#9b8cff`
- Success: `#59c98f`
- Warning: `#f6b84b`
- Error: `#ef6f73`
- Background: `#fffaf5`
- Surface: `#ffffff`
- Text primary: `#20313f`
- Text secondary: `#697887`

The web theme is implemented in `frontend-web/src/theme.ts`. The mobile theme is implemented in `mobile-app/src/theme.ts`.

## Components

The web app exposes a `/ui-kit` route with buttons, inputs, metric cards, recommendation cards, notifications, tabs, progress indicators, alerts, modals, and loading states.
