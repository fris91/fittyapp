FittyClean UI · alt style

UX strategy

A calm, encouraging companion — not a dashboard you operate. Every screen answers one question: “what’s one good thing I can do right now?”

Design principles

  • One screen, one job. Progressive disclosure over density — detail lives a tap deeper.
  • Encourage, don’t grade. Soft language, celebrate streaks, never shame a missed day.
  • Fast in, fast out. Logging anything is ≤2 taps via the center button + bottom sheets.
  • Wellness ≠ medical. Clinical-looking numbers always carry a quiet “not medical advice” boundary.
  • Trustworthy AI. Every plan is editable, explainable, and shows what data it used.
  • Soft & physical. Color carries meaning — coral = action, mint = positive, lavender = AI, amber = attention.

Mobile navigation

Five-tab bottom nav with a raised center “+” for instant logging.

🏠 Today📈 Progress＋ Log🥗 Plans✨ Coach

The daily loop

Built around three tiny touchpoints so even a 15-second visit feels complete:

  • Morning check-in — one tap on mood/energy + today’s focus card.
  • Midday log — one meal or workout via ＋; instant positive feedback + rings fill.
  • Evening reflection — Coach summarizes the day and previews tomorrow.

Three home concepts

A · Data
The cockpit

Metric grid + trends up front. Powerful for quantified-self users, but can feel clinical.

B · AI coach
The companion

Conversation leads, data one tap away. Warmest & lowest-effort; needs a strong coach.

★ C · Habit
The momentum board

Single focus + streak/rings + inline coach voice. Best for the broad “be healthy” user — recommended.

Recommendation: ship Concept C as default, borrow B’s coach voice inline, expose A’s full grid under Progress.
